Start Menu

The start menu of the Glowing Pong game is designed to be sleek and user-friendly. Upon launching the game, players are presented Quit Game The "Quit Game" option in Glowing Pong allows players to exit the game and return to their desktop or main device interface. When selected, this option prompts a confirmation dialogue to ensure that the player intends to quit the game. Upon confirmation, the game closes gracefully, providing a smooth transition out of the gaming experience. The main menu offers convenient navigation and control over the game experience, empowering players to tailor their gameplay and access essential information effortlessly.

Points Table

The Glowing Pong game features a dynamic points system. Players earn points by successfully hitting the ball past their opponent's paddle. The game tracks and displays each player's score on the screen in real-time, fostering a sense of competition and encouraging players to strive for higher scores. Winning a match requires reaching a predetermined score limit, adding a layer of strategy and excitement to each game session.

What I Have Done

In developing the Glowing Pong game for my portfolio, I undertook several key responsibilities: Designed and implemented game mechanics, including paddle movement, ball physics, and collision detection. Developed the start menu interface and game navigation for seamless player interaction. Programmed the scoring system to accurately track and display player scores throughout the game. Integrated dynamic visuals and glowing neon effects to enhance the game's aesthetic appeal and visual feedback. Tested and optimized gameplay for various platforms to ensure a smooth and enjoyable gaming experience.

Gameplay

During gameplay, players control their paddles using keyboard or touchscreen inputs, depending on the platform. The glowing ball moves between the paddles, bouncing off walls and paddles with realistic physics. Players must anticipate the ball's trajectory and use precise timing to return it to their opponent while avoiding letting the ball slip past their own paddle. The game's intensity increases as players score points, with the ball speeding up and adding to the challenge.

End of Game

The Glowing Pong game concludes when one player reaches the score limit, signaling victory. Upon game completion, players are presented with a game over screen displaying the winner and offering options to start a new game or return to the main menu. The end of the game provides closure to the gameplay session while inviting players to continue enjoying the game.
